Game Mode Flow


On BeginPlay, the game mode begins a timer for round length. As players join, they are added to an array of Player Controllers and a server message is displayed announcing their arrival. New players are run through a custom spawn function which uses an interface to set some pawn variables based on server and mode settings; this will be more useful when multiple classes are enabled. On a kill or CTF capture, the game mode is alerted and player scores are incremented within their player states, which are then replicated. The respawn function is bound to a DeathCaller function in the Player Controller. When the round is over, the game mode sets a "game is over" bool to prevent respawns.
